# Digital Notice Board 📰

A full-stack web application to manage and display notices digitally.

## 🚀 Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB

## ✨ Features

* Admin Login
* Create, Read, Update, Delete Notices
* Search Functionality
* Pagination
* Pin Important Notices

## 📂 Project Structure

* frontend/
* backend/

## ⚙️ Setup Instructions

### 1. Clone the repo

```bash
git clone https://github.com/thamarailogu/digital-notice-board.git
```

### 2. Install dependencies

```bash
cd frontend
npm install

cd ../backend
npm install
```

### 3. Run project

```bash
# backend
node server.js

# frontend
npm run dev
```
